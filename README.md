# 🎓 Student Management System (SQL)

A SQL-based **Student Management System** built using **MySQL** that demonstrates database design, table relationships, CRUD operations, SQL joins, aggregate functions, and report generation.

This project is designed to strengthen SQL fundamentals by implementing a relational database from scratch and performing various data manipulation and analysis operations.

---

## 📌 Features

* Create a relational database
* Design tables with Primary Keys and Foreign Keys
* Insert sample data
* Perform CRUD (Create, Read, Update, Delete) operations
* Execute INNER JOIN, LEFT JOIN, and RIGHT JOIN queries
* Generate reports using Aggregate Functions
* Modify table structure using ALTER TABLE
* Maintain data integrity using constraints

## 🗄 Database Schema

The project consists of three tables:

### Student

Stores student information such as:

* Student ID
* Student Name
* Age
* City
* Email

### Course

Stores course information such as:

* Course ID
* Course Name
* Course Fee

### Enrollment

Acts as a bridge table between Student and Course.
It stores:

* Enrollment ID
* Student ID (Foreign Key)
* Course ID (Foreign Key)
* Enrollment Date
